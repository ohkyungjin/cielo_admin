# 관리자 시스템 섹션별 상세 구조

## 1. 대시보드

### 1.1 예약 현황 위젯
- 당일 예약 현황 카운터
- 진행중인 예약 목록
- 대기중인 예약 알림
- 예약 상태별 퀵 필터

### 1.2 매출 현황 위젯
- 일간/주간/월간 매출 그래프
- 전월 대비 증감률
- 목표 달성률
- 서비스별 매출 분포

### 1.3 알림 센터
- 미처리 알림 목록
- 알림 우선순위 표시
- 알림 바로가기 링크
- 알림 읽음 처리

## 2. 예약 관리

### 2.1 예약 목록
- 필터링 옵션
  - 날짜 범위
  - 예약 상태
  - 서비스 종류
  - 담당자
- 검색 기능
  - 예약번호
  - 보호자명
  - 반려동물명
  - 연락처
- 목록 표시 항목
  - 예약번호
  - 예약일시
  - 보호자명
  - 반려동물명
  - 서비스 종류
  - 상태
  - 금액

### 2.2 예약 생성/수정
#### Step 1: 보호자 정보
- 필수 입력 사항
  - 이름
  - 연락처
  - 이메일
- 선택 입력 사항
  - 주소 (카카오 주소 API)
  - 방문 경로
  - 특이사항

#### Step 2: 반려동물 정보
- 기본 정보
  - 이름
  - 품종
  - 나이
  - 성별
  - 몸무게
- 사망 관련 정보
  - 사망일
  - 사망 경위
  - 장례확인서 발급 여부
- 추가 정보
  - 경유업체
  - 특이사항

#### Step 3: 예식 선택
- 예식 패키지 선택
  - 기본 예식
  - 고급 예식
  - 최고급 예식
  - 호화 예식
- 옵션 그룹 선택
  - 생화 세트
  - 관 세트
  - 유골함 세트
  - 이불 세트
  - 액자 세트
- 무게별 추가 비용
  - 5Kg 이하: 0원
  - 10Kg 이하: 10만원
  - 15Kg 이하: 20만원
  - 15Kg 초과: 입력
- 할인 설정
  - 할인율(%)
  - 할인금액

#### Step 4: 최종 확인
- 예약 정보 요약
- 결제 정보
- 디지털 서명
- PDF 출력
