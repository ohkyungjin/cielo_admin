2. 예약 관리

구성 요소

예약 리스트

테이블 형태: 예약번호, 보호자명, 반려동물명, 예약일.

필터: 날짜별, 상담 상태별 필터링 기능 제공.

상세 페이지

상담을 진행하면서 단계별로 입력하도록 Step 형태로 구현:

1단계: 보호자 정보

보호자 이름, 연락처, 이메일, 주소(API이용).

2단계: 반려동물 정보

반려동물 이름, 품종, 나이, 성별(암/수), 몸무게(Kg), 사망일, 사망경위(자연사, 병사, 사고사, 안락사), 방문경로(인터넷, 블로그, 병원, 지인소개), 경유업체, 장례확인서(발급/미발급).

3단계: 예식 선택 및 금액 확인

예식 선택:

선택 가능한 4가지 예식(예: 기본예식, 고급예식, 최고급예식, 호화예식)을 표시.

예식 별 옵션 그룹을 보여주고, 선택 가능한 옵션 표시 ( 그룹 별 옵션은 하나만 선택 가능 )

각 예식에 포함된 이미지와 서비스 가격 상세 정보 제공.

무게 별 추가 비용:

5Kg 이하 : 0원

10kg 이하 : 10만원

15Kg 이하 : 20만원

15Kg 이상 : 금액입력필드 제공

할인 적용:

할인율(%) 또는 금액 입력 필드.

최종 금액 계산:

예식 선택과 할인 금액 적용 후 최종 금액을 실시간으로 계산해 화면 하단에 표시.

4단계: 최종 예약 현황 및 서명

예약 확인:

입력된 보호자 정보, 반려동물 정보, 선택한 예식 및 옵션 및 최종 금액을 요약 형태로 표시.

디지털 서명:

react-signature-canvas를 사용해 보호자와 관리자가 서명할 수 있는 캔버스 제공.

저장 및 완료 버튼:

"예약 완료" 버튼 클릭 시 모든 데이터 저장 및 예약 확정.

알림 전송

상담 상태 변경 또는 예약 확정 시 SMS/이메일 자동 발송:

알림 내용: 예약 번호, 예약일, 선택한 예식, 최종 금액.

CSV 다운로드

예약 데이터를 CSV 파일로 다운로드 가능.

단계 진행 상태 표시

각 단계 진행 시, 현재 진행 중인 단계를 시각적으로 표시하는 프로그레스 바 추가.

자동 데이터 검증

각 단계별 필수 입력 항목이 모두 완료되어야 다음 단계로 이동 가능.
