# 관리자 시스템 상세 유저 플로우

## 1. 인증 프로세스

### 1.1 로그인
1. 로그인 페이지 접속
2. 로그인 방식 선택
   - 일반 로그인: 아이디/비밀번호 입력
   - 소셜 로그인: 카카오/구글 선택
3. 자동 로그인 옵션 선택
4. 인증 처리
5. 대시보드로 이동

### 1.2 회원가입
1. 회원가입 페이지 접속
2. 가입 방식 선택
3. 기본 정보 입력
4. 이메일 인증
5. 가입 완료

## 2. 예약 관리 프로세스

### 2.1 예약 목록
1. 예약 관리 페이지 접속
2. 필터 적용 (날짜/상태)
3. 예약 검색
4. 상세 보기/수정/삭제 작업

### 2.2 새 예약 생성
1. 보호자 정보 입력
   - 이름, 연락처, 이메일 입력
   - 주소 검색 및 입력
2. 반려동물 정보 입력
   - 기본 정보: 이름, 품종, 나이, 성별, 몸무게
   - 사망 관련: 사망일, 사망경위
   - 추가 정보: 방문경로, 경유업체
3. 예식 선택
   - 예식 패키지 선택
   - 옵션 그룹별 선택
   - 무게별 추가 비용 계산
   - 할인 적용
4. 최종 확인
   - 정보 확인
   - 디지털 서명
   - 예약 완료
5. 자동 알림 발송

## 3. 서비스 관리 프로세스

### 3.1 예식 패키지 관리
1. 서비스 목록 확인
2. 새 패키지 추가/기존 패키지 수정
3. 기본 정보 설정
4. 가격 정책 설정
5. 활성화 상태 관리

### 3.2 옵션 관리
1. 옵션 그룹 관리
   - 그룹 생성/수정
   - 순서 설정
   - 필수 여부 설정
2. 개별 옵션 관리
   - 옵션 추가/수정
   - 가격 설정
   - 이미지 업로드

## 4. 통계 및 리포트

### 4.1 통계 확인
1. 기간 선택
2. 통계 유형 선택
3. 데이터 필터링
4. 그래프 확인
5. 데이터 추출

### 4.2 리포트 생성
1. 리포트 유형 선택
2. 기간 설정
3. 데이터 집계
4. PDF/CSV 생성
5. 다운로드

## 5. 추모 관리

### 5.1 게시글 관리
1. 게시글 목록 확인
2. 새 게시글 작성
3. 이미지 업로드
4. 내용 입력
5. 게시 승인
6. 웹사이트 연동
